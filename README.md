# Hazel Han Portfolio

Static portfolio website for overseas operations, brand marketing, advertising planning, and content growth applications.

## Files

- `index.html` is the website content.
- `styles.css` controls the visual design.
- `assets/` contains images, CVs, selected decks, the Stardew Valley video, and the FREQUENCY: KOREA report.
- The site is Chinese-first with English role keywords for China-based overseas operation applications.

## Publish on GitHub Pages

1. Create a GitHub repository named `hazel-portfolio` or `yourusername.github.io`.
2. Upload everything inside this folder.
3. Go to repository Settings, then Pages.
4. Choose `Deploy from a branch`, select `main`, and use `/root`.
5. Wait for GitHub to generate the site URL.

## Before sending to recruiters

- Replace the LinkedIn/contact information if needed.
- Keep only polished and non-confidential files in `assets/`.
- For commercial projects, keep sensitive numbers and internal documents summarised.
- `stardew-ccf-campaign.pptx` is a large file. GitHub can host it, but if upload is slow, move it to cloud storage and replace the link in `index.html`.
- `frequency-korea-report.pdf` is linked from the cultural activation case and should stay in `assets/`.
